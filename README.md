# Social App

This repository contains a simple MERN-stack social media application with user authentication, posts, and notifications.

## Folders
- `backend` – Express server with MongoDB models and routes.
- `frontend` – React application built with Vite.

Both apps have their own `package.json` files with dependencies and scripts.

### Frontend Pages
- `/login` – login form
- `/signup` – registration form powered by `SignupForm` component
